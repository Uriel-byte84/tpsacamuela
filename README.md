# Sistema de Gestión - Consultorio Odontológico "Saca Muela"

Este proyecto consiste en un sistema de gestión interno para un consultorio odontológico, desarrollado bajo una arquitectura funcional CRUD y separación estricta de capas en Python utilizando la librería gráfica `tkinter`.

## 📌 Estructura del Proyecto
*   **Hito 0:** Especificación de requerimientos funcionales, no funcionales y de dominio.
*   **Hito 1:** Diseño del sistema con Diagrama de Clases (UML) modelado en Draw.io.
*   **Hito 2:** Implementación del código fuente:
    *   `main.py`: Punto de entrada de la aplicación.
    *   `vistas.py`: Interfaz gráfica con solapas (`ttk.Notebook`) para el registro y listado.
    *   `modelos.py`: Clases del sistema (`Persona`, `Paciente`, `Odontologo`).
    *   `base de datos.py`: Configuración y consultas mediante `sqlite3`.
